1. **Clone this repo:**
    ```
    git clone https://github.com/sairampolisetty/imagineZY.git
    ```

2. **Backend Setup**
    ```

    ```

    - Frontend at `http://localhost:5173`
    - Backend at `http://localhost:8080` (or as configured)
